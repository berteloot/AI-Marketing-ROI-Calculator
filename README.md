# Nytelligence ROI - Marketing AI Workflow ROI Calculator

A production-grade web application that calculates ROI for marketing AI workflows using OpenAI-powered benchmarks and deterministic financial calculations.

## Features

- **AI-Powered Workflow Analysis**: Uses OpenAI function calling to break down marketing workflows into tasks and suggest automation coverage
- **Deterministic ROI Calculations**: Strict mathematical formulas ensure no AI hallucinations in financial metrics
- **Multiple Scenarios**: Compare Base, Conservative, and Aggressive projections
- **Business Case Generator**: Automatically generates CFO-ready summaries and pilot pitches
- **Workflow Visualization**: Charts and tables showing task breakdown and AI coverage

## Tech Stack

- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **OpenAI API** (GPT-4o with function calling)
- **Recharts** (data visualization)

## Setup

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Set up environment variables**:
   Create a `.env.local` file in the root directory:
   ```env
   OPENAI_API_KEY=your_openai_api_key_here
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Open your browser**:
   Navigate to [http://localhost:3000](http://localhost:3000)

## How It Works

### 1. User Input
Users select:
- Marketing Department Type (B2B SaaS, Agency, etc.)
- Primary Workflow (LinkedIn content, Email nurture, etc.)
- Team Size
- Average Hourly Cost
- Industry (optional)

### 2. AI Benchmark Generation
OpenAI function calling:
- Breaks down the workflow into specific tasks
- Estimates hours per task
- Suggests AI coverage percentages
- Recommends efficiency gains
- Suggests relevant AI tools with pricing
- Optionally suggests revenue impact models

### 3. Deterministic ROI Calculation
The app uses strict mathematical formulas (no AI) to calculate:
- Labor cost savings
- Tech costs (licensing + training)
- Revenue impact (if applicable)
- ROI percentage
- Payback period
- 3-month pilot savings
- Risk-adjusted savings

### 4. Scenario Comparison
Three scenarios are generated:
- **Conservative**: 70% of base efficiency gains
- **Base**: 100% of AI-suggested efficiency gains
- **Aggressive**: 130% of base efficiency gains

### 5. Business Case Generation
Automatically generates:
- CFO summary
- Pilot pitch
- Slide bullets
- Full business case document

## Key Design Principles

1. **AI for Inputs, Math for Outputs**: OpenAI generates workflow breakdowns and estimates, but all ROI calculations use deterministic formulas
2. **No Hallucinations**: Financial metrics are never generated by AI
3. **Production-Ready**: Built for CMOs and CFOs with credible, auditable calculations
4. **Marketing-Specific**: Focused on marketing workflows, not enterprise-wide transformation
